# 🏗️ SQL Data Warehouse Project

A complete **Data Warehouse and Analytics solution** built using **SQL Server** and **Medallion Architecture** (Bronze → Silver → Gold). This is a portfolio project showcasing my skills in data engineering, ETL pipelines, and data modeling.

---

## 📐 Data Architecture

| Layer | Description |
|-------|-------------|
| 🥉 **Bronze** | Raw data ingested as-is from CSV source files into SQL Server |
| 🥈 **Silver** | Cleaned, standardized, and normalized data ready for analysis |
| 🥇 **Gold** | Business-ready data modeled into a Star Schema for reporting |

---

## 📖 What I Built

- ✅ Medallion Architecture — Bronze, Silver, Gold data layers
- ✅ ETL Pipelines — Stored procedures to extract, transform, and load data
- ✅ Star Schema — Fact and dimension tables optimized for analytics
- ✅ Data Quality Checks — Validation scripts for Silver and Gold layers
- ✅ SQL Analytics — Insights on customer behavior, product performance, and sales trends

---

## 🛠️ Tools & Technologies

- SQL Server Express
- SSMS (SQL Server Management Studio)
- T-SQL
- Git & GitHub

---

## 📂 Repository Structure

scripts/bronze/ — Raw data ingestion
scripts/silver/ — Data cleaning and transformation
scripts/gold/   — Star Schema analytical models
tests/          — Data quality checks

---

## 🚀 How to Run

1. Install SQL Server Express + SSMS
2. Clone this repo
3. Run scripts/gold/init_database.sql first
4. Run Bronze → Silver → Gold scripts in order
